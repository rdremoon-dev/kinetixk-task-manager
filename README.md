# Kinetixk 🚀

Kinetixk is a sleek, lightweight, and modern task management application built using vanilla frontend technologies. Featuring a responsive glassmorphism user interface, dynamic contextual actions, intelligent sidebar tracking filters, and real-time client-side persistence, it is designed to keep you productive and organized seamlessly.

🌐 **Live Demo:** [https://kinetixk.netlify.app](https://kinetixk.netlify.app)

---

## ✨ Features

* **Sleek Glassmorphism UI:** A visually stunning backdrop mesh gradient paired with blurred frosted-glass container containers.
* **3-Dot Contextual Action Menus:** Edit details or permanently delete individual tasks directly from an interactive overlay menu built on each task card.
* **Smart Local Persistence:** Integrated synchronization with browser `localStorage`. Your active and completed tasks will survive full browser cache refreshes safely.
* **Context Sidebar Filters:** * **Inbox:** Your master task control list pipeline.
  * **Today:** Displays only critical items scheduled for execution immediately.
  * **Upcoming:** Maps forward horizons, filtering upcoming priorities.
  * **Filters & Labels:** Automatically filters and sorts your active tasks by Priority Weight ($P1 \rightarrow P4$).
* **Fluid Checkbox Interaction:** Fully styled custom circular task checkboxes that leverage a clean fluid horizontal sliding exit transition once ticked.
* **Hidden Scroll Mechanics:** Main task lists scroll natively while completely hiding default browser scrollbars for a clean workspace look.

---

## 🛠️ Built With

* **HTML5:** Semantic document structuring.
* **CSS3:** Custom properties, CSS Grid/Flexbox layouts, glassmorphic filters, and interactive transition states.
* **Vanilla JavaScript (ES6+):** Dynamic card rendering engine, state data pipeline management, filter handlers, and `localStorage` synchronization.
* **Remix Icons:** Premium iconography packages loaded asynchronously over CDN.
